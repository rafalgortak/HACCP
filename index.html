Admin — lokalizacje

Dodajesz nowe lokale, kuchnie lub punkty sprzedaży. Po dodaniu lokalizacja pojawi się w formularzach, raportach i uprawnieniach pracowników.

Admin — urządzenia

Tu modyfikujesz listę chłodni i mroźni dla każdej lokalizacji.

Admin — pora dnia / etap zmiany

Dla każdej lokalizacji możesz dodać własne etapy, np. Otwarcie, W ciągu dnia, Zamknięcie zmiany.

Admin — sprzątanie

Dodajesz i usuwasz obowiązki sprzątania osobno dla: rano / w ciągu dnia / wieczór.

Admin — HACCP / daty

Dodajesz nowe kontrole HACCP, gdy pojawią się nowe obowiązki.

Admin — pracownicy i dostęp

Dodajesz pracownika, PIN, rolę i lokalizacje.

Przypisanie lokalizacji:

Edycja zapisanych wpisów

Przegląd i edycja wpisów są teraz w osobnej zakładce, żeby panel Admin nie ładował długiej listy.

Admin — backup i migracja do Firebase

Przed przejściem na Firebase zrób kopię danych. Ten plik będzie można później zaimportować do wersji chmurowej.

Zakresy

Chłodnie: 0–10°C. OK 0–4, UWAGA 5–7, ALARM 8–10.

Mroźnie: -25 do -5°C. OK -25 do -18, UWAGA -17 do -15, ALARM -14 do -5.
